## 🚀 **QUICK START GUIDE**

### **📦 INSTALLATION**
```bash
# Clone repository
git clone [repository-url]
cd DobbyX

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your Discord and Fireworks API credentials

# Deploy commands to Discord
npm run deploy

# Start the bot
npm start
```

### **🔧 ENVIRONMENT SETUP**
```bash
# Required Environment Variables
DISCORD_TOKEN=[Your Discord Bot Token]
DISCORD_CLIENT_ID=[Your Discord Application ID]
GUILD_ID=[Your Discord Server ID]
FIREWORKS_API_KEY=[Your Fireworks API Key]
DOBBY_MODEL_ID=accounts/sentientfoundation/models/dobby-unhinged-llama-3-3-70b-new
```

### **🎯 FIRST DEPLOYMENT**
1. Create Discord application and bot
2. Get Fireworks API access for SentientAGI model
3. Configure environment variables
4. Deploy commands: `npm run deploy`
5. Start bot: `npm start`
6. Verify all 5 real-time systems initialize
